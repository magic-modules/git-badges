@magic-modules/git-badges

this is the @magic-modules GitBadges component. shows git badges for a github/gitlab repository.

installation

installation is done using npm. for now, all magic modules are living on github and not on npm.

note the missing @ before magic-modules.

this is how we install npm modules from github.

usage

in a page/component, just pass the badge user/project string for those badges you want to show:

gitlab / other git hosts

to use gitlab or any other git host in the urls that need a git host to be set, simply pass a host prop to the GitBadges module
